# Text Analysis & Editing Center (TextAEC)

TextAEC is a web-based text editor and analysis tool designed to provide advanced text editing, formatting, and real-time analysis. It is built with **React** and supports multiple export formats, word frequency analysis, voice typing, and text-to-speech.

---

## üìù Features

- **Text Editing**
  - Undo/Redo
  - Case transformations (Upper, Lower, Title, Sentence)
  - Remove extra spaces or line breaks
  - Sort words alphabetically
  - Character limit and font customization

- **Text Analysis**
  - Count characters, words, unique words, sentences, and paragraphs
  - Calculate average word and sentence length
  - Live word frequency word cloud

- **Highlight & Search**
  - Highlight specific words with custom colors
  - Find & Replace functionality

- **Export**
  - Download text as `.txt`, `.pdf`, `.docx`, `.csv`, or `.json`

- **Voice Features**
  - Text-to-Speech
  - Voice Typing with multiple languages (English, Bangla, Hindi, Spanish, French)

- **Live Preview**
  - Preview highlighted text in real-time

- **Keyboard Shortcuts**
  - `Ctrl+Z` ‚Üí Undo
  - `Ctrl+Y` ‚Üí Redo
  - `Ctrl+S` ‚Üí Save TXT
  - `Ctrl+P` ‚Üí Save PDF
  - `Ctrl+D` ‚Üí Save DOCX
  - `Ctrl+F` ‚Üí Focus on search
  - `Ctrl+R` ‚Üí Replace
  - `Ctrl+L` ‚Üí Reset all
